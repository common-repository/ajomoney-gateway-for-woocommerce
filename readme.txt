Provide AjoMoney as a payment option for WooCommerce orders. Either pay in full or pay later.

== Description ==
Give your customers the option to buy now and pay later or even pay in full with AjoMoney. The “AjoMoney Gateway for WooCommerce” plugin provides the option to choose AjoMoney as the payment method at the checkout. For each payment that is approved by AjoMoney, an order will be created inside the WooCommerce system like any other order.

== Installation ==
This section outlines the steps to install the AjoMoney plugin.

> Please note: If you are upgrading to a newer version of the AjoMoney plugin, it is considered best practice to perform a backup of your website - including the WordPress database - before commencing the installation steps. AjoMoney recommends all system and plugin updates to be tested in a staging environment prior to deployment to production.

> The plugin only supports Naira (NGN) currency.

1. Login to your WordPress admin.
1. Navigate to \"Plugins > Add New\".
1. Type \"AjoMoney\" into the Keyword search box and press the Enter key.
1. Find the \"AjoMoney Gateway for WooCommerce\" plugin. Note: the plugin is made by \"AjoPay Financial Technology Limited\".
1. Click the \"Install Now\" button.
1. Click the \"Activate\" button.
1. Navigate to \"WooCommerce > Settings\".
1. Click the \"Checkout\" tab.
1. Click the \"AjoMoney\" sub-tab.
1. Enter the Secret Key that was provided by AjoMoney for your business for Production use.
1. Save changes.
